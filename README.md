# 💻 Projeto: Conta Bancária em Java (POO)

Este projeto simula uma conta bancária simples utilizando conceitos de **Programação Orientada a Objetos(POO)** com a linguagem Java.

## 🗂️ Estrutura do Projeto

O projeto foi dividido em duas versões para mostrar minha evolução:

### 🆚 Versão 1 
- Criação da classe `Conta`
- Métodos para depósito, saque e exibição de dados
- Uso de **encapsulamento** 

### 🆚 Versão 2
- Evolução da versão 1
- Uso de **herança**, criando a subclasse `ContaPoupanca`
- Organização por pacotes(`v1` e `v2`)
